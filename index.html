Tutorial

W = Cima

S = Baixo

A = Esqueda

D = Direita

Botão esquerdo do mouse = Tiro



Movimente-se dentro do retângulo branco e evite encostar o quadrado azul(Player) no quadrado vermelho(Inimigo).

Ao atirar o campo de dano na cor amarela fica ativo por 3 segundos, caso algum Inimigo passe por esse campo ele sofrerá dano.

Seu objetivo é eliminar todos os Inimigos para passar de nível.

Conforme for aumentando os níveis vai aparecer mais inimigos com mais velocidade e mais resistência.

Atenção!

Para que o jogo funcione corretamente é necessário colocar a resolução do seu computador em 1920 X 1080 pixels.

Caso já esteja nessa resolução ignore esta mensagem, este jogo receberá atualizações para deixar sua interface mais responsiva.
